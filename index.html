Aleks' Cheatsheet
Tag Name Description
Table tags
<table> Table The wrapper element for all HTML tables.
<thead> Table head The set of rows defining the column headers in a table.
<tbody> Table body The set of rows containing actual table data.
<tfoot> Table footer The set of rows defining the footer in the table.
<tr> Table row The table row container.
<td> Table data The data contained in the table.
Table attributes
colspan Column span Defines how many columns a td element should span.
rowspan Row span Defines how many rows a td element should span.
Skeleton table
<table>
 <thead>
  <tr>
    <th>insert header col 1 text</th>
    <th>insert header col 2 text</th>
    <th>insert header col 3 text</th>
  </tr>
 </thead>
 <tbody>
  <tr>
    <td>insert data col 1 here</td>
    <td>insert data col 2 here</td>
    <td>insert data col 3 here</td>
  </tr>
 </tbody>
 <tfoot>
  <tr>
    <td>insert footer col 1 here</td>
    <td>insert footer col 2 here</td>
    <td>insert footer col 3 here</td>
  </tr>
 </tfoot>
</table>
